Luganda)\n\n```\n1. USER ACTION\n   └─ Clicks \"Translate\" button on post\n\n2. FRONTEND (JavaScript)\n   └─ fetch('/hotel/gemini-translate/', {\n        method: 'POST',\n        body: {\"text\": \"Good morning\", \"target_language\": \"lg\"}\n      })\n\n3. DJANGO BACKEND\n   └─ gemini_translate() view receives request\n      ├─ Parses JSON\n      ├─ Calls translate_smart()\n      └─ translate_smart():\n         ├─ Checks cache: miss\n         ├─ Is \"lg\" in NLLB_LANGS? YES ✅\n         ├─ POST to NLLB API\n         ├─ NLLB responds: \"Wasuze otya\"\n         ├─ Cache for 7 days\n         └─ Return to Django view\n\n4. DJANGO RESPONSE\n   └─ JsonResponse({\n        'success': True,\n        'translated': 'Wasuze otya',\n        'target_language': 'lg',\n        'cached': False\n      })\n\n5. FRONTEND (JavaScript)\n   └─ Display translated text to user\n      \"Good morning\" → \"Wasuze otya\"\n\n6. CACHE (7 days)\n   └─ Next user requests same text in Luganda?\n      Instant return from cache! ⚡\n```\n\n---\n\n## Performance Timeline\n\n```\n                    FIRST REQUEST (NLLB Cold Start)\n├─────────────────────────────────────────────────────────┤\n0s              5s              10s              15s    30s ✅\n├────────────────┤\nRequest sent\n                ├───────────────┤\n              Model loading\n              (1.2GB from disk)\n                         ├───────┤\n                      Translate\n                      (inference)\n                                    ├─────┤\n                                  Network\n                                  response\n                                        └───────► Result\n\n               SECOND REQUEST (Warm Model)\n├────┤\nRequest sent\n    ├────┤\n  Response\n\n               THIRD+ REQUEST (Cached)\n├─┤\nFetched from\nDjango cache\n```\n\n**Timeline:**\n- First NLLB request: 20-30 seconds\n- Second NLLB request: 400-500ms (model already loaded)\n- LibreTranslate request: 200-300ms (always)\n- Cached request: <10ms\n\n---\n\n## Error Handling Flow\n\n```\n                    Translation Request\n                           ↓\n                    Check Cache\n                           ↓\n                      Cache HIT?\n                      YES→ Return ✅\n                      NO↓\n                    Try NLLB?\n                      YES↓\n                  NLLB Success?
